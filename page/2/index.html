基础

语言特性

  • 所有的东西都是对象（变量var、数字numbers、函数function、null等）。

    所有的对象都是类的实例。

    所有的对象都继承自内置的 Object 类。

  • Dart 动态类型语言，尽量给变量定义一个类型，会更安全，没有显示定义类型的变量在debug模式下类型会是dynamic（动态的）

  • 程序中指定数据类型使得程序合理地分配内存空间，并帮助编译器进行语法检查。

    Dart 是弱数据类型。

  • Dart 代码在运行前解析所有的代码。

    指定数据类型和编译时的常量，可以提高运行速度。

  • Dart 中类和接口时统一的，类即接口，可以继承一个类也可以实现一个类（接口），自然也包含了良好的面向对象和并发编程的支持。

  • Dart 程序有统一的程序入口：main()

    这点与 C/C++ 语言相像。

    Dart 提供了顶级函数（如main()）

  • Dart 没有初始化的变量默认值null

Vim常用命令

进入vi的命令

命令 说明
vi filename 打开或新建文件，并将光标置于第一行首
vi +n filename 打开文件，并将光标置于第n行首
vi + filename 打开文件，并将光标置于最后一行首
vi +/pattern filename 打开文件，并将光标置于第一个与pattern匹配的串处
vi -r filename 在上次正用vi编辑时发生系统崩溃，恢复filename
vi filename….filename 打开多个文件，依次进行编辑
