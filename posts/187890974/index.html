验证nginx配置文件是否正确

法一：进入nginx安装目录sbin下，输入命令./nginx -t
看到如下显示nginx.conf syntax is ok
nginx.conf test is successful说明配置文件正确！

法二：在启动命令-c前加-t
进入nginx安装目录sbin下(/user/local/nginx/sbin/nginx)，输入 -t -c / /usr/local/nginx/conf/nginx.conf

重启Nginx服务

法一：进入nginx可执行目录sbin下，输入命令./nginx -s reload 即可

法二：查找当前nginx进程号（ps -ef|grep nginx），然后输入命令：kill -HUP 进程号 实现重启nginx服务
