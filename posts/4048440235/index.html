Shift+Enter 在当前行下插入一行

撤销 Ctrl+Z
前进 Ctrl+Y

eclipse书签的使用

窗口管理所有书签

先打开书签：在eclipse平台上，window–>Show View –>Other..–>General–>Bookmarks

添加书签

法一：单击菜单栏上的Edit—–>Add Bookmark…—>然后填写书签名

法二：在编辑区最左边，也就是行号的左边（也是设定断点的地方），点击右键，在弹出的菜单里选择Add Bookmark—>然后填写书签名;

设置

设置代码、控制台背景色

代码区：Windows->Preferences->Editors->Text Editors->更改Background color

color(255，245，216)

控制台：在控制台Console右键

设置字体大小

Windows->Preferences->General->Appearance->Colors and Fonts->Basic->Text Font

查看源代码

window->perferences->java->Installed->JRE，双击出现的jdk1.7.0_80（或双击出现的jre）

在出现的弹窗中选中rt.jar，选择Source Attachment...，在新弹窗中看External location下的Path框内应该关联到C:/Program Files/JAVA/jdk1.7.0_80/src.zip，这样就OK了。

jsp、html标签对选中时变色

Window--Preferences--General--Editors--Text Editors--Annotations：找到Matching Tags，修改右侧的Colors

  3. Timeouts-Start：修改启动的超时时间

  4. Ports-HTTP/1.1：修改服务的端口号

出错 “严重: PWC6117”

查了网上说的是

但是我代码里没有这个问题。

原因：通过断点发现连接的数据库不对，而代码对少了个error_login的错误码的登陆失败情况的处理

解决：更改数据库连接；增加error_login错误码的处理。

源码路径错误

A cycle was detected in the build path of project ‘EasyH5’….

解决：

  1. Eclipse/STS -> Window -> Preferences… -> Java -> Compiler -> Building -> Building path problems -> Circular dependencies -> 将Error改成Warning
  2. 或者查看工程之间是否真的需要循环引用，不需要的话去掉。
