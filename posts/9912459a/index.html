设置更改

编辑器

  1. 显示
    缩减栏宽：关闭【让界面显示内容更多】
    折叠标题：打开
    显示缩进参考线：关闭【让列表旁边的参考线消失】

  2. 行为
    使用制表符：关闭【用tab键的时候会插入空格】

文件与链接

  • 始终更新内部链接：打开
  • 新建笔记的存放位置：当前文件所在的文件夹
  • 内部链接类型：基于当前笔记的相对路径
  • 使用Wiki链接：关闭【关闭后会使用markdown的[]()语法】
  • 检测所有类型文件：打开【会罗列出所有的文件类型，即使obsidian不支持这种文件类型】
  • 附件默认存放路径：当前文件所在文件夹下指定的子文件夹中
    子文件夹名称：img

  • Interface

  • 高级
    • 原生菜单：打开

第三方插件

安装

  • Reveal Active File Button plugin 【切换右侧笔记，左侧自动定位到文件位置】
  • Auto Link Title 【自动补全插入链接的标题】
  • Templater 【模板】
    要先创建个文件，在新文件中使用：Alt+E
  • Paste image rename
    命名规则：..\.\/img\/{{fileName}}_{{DATE:YYYYMMDDHHmmss}} 要与 附件默认存放路径改成img 结合使用
  • Floating TOC 【显示大纲】
  • style settings 【可优化界面】
    最后记得在设置页面的最底部把这些插件打开才会生效

    [!tips]
    快捷键：Alt+E

  • Code Styler【优化代码样式】

    设置代码块折叠：需要在代码块首行添加 fold，若要显示内容可以 fold title:这是标题，如果带空格则 fold title:"这是 标题"
