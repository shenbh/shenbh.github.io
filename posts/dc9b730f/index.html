基本类型

  • 数字

    • Double（64）

    • Float（32）

    • Long（64）

    • Int（32）

    • Short（16）

    • Byte（8）

  • 字符串

    • val s = “i = $i”

    • val s = “i = ${i+5}”

  • 变量与常量

    • 常量 val a : Int = 1

    • 变量 var a : Int = 1

属性包装

  • 开发者不需要显式地声明属性的 get/set 方法了

函数（fun）

  • 单表达式 fun add(a: Int, b: Int): Int = a + b

  • 返回值 – 默认返回值 Kotlin.Unit

  • 函数类型

    • 顶级函数

    • 类成员函数（默认是 public final 类型的，无法被声明成 static 类型）

    • 对象函数（声明在对象中的函数，如伴随对象，单例对象）

    • 本地函数（声明在其他函数内部）

      • 高阶函数

      • 内联函数

      • 扩展函数

      • 匿名函数

    • 函数入参

      • 不定长入参

        • java方式

          • String…msg

        • Kotlin方式

          • vararg t : Int

    • 闭包（局部函数）

      • 概念：在局部函数内，可以访问宿主和类内部的资源，这些资源包括数据和函数

      • 作用：突破了函数内的数据随着堆栈销毁而被销毁

      • 闭包的作用域与返回

    • lambda表达式

      • 函数入参与返回值

      • 函数实例化与lambda表达式
