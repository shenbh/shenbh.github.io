jQuery教程

jQuery 选择器

语法 描述
$(this) 当前HTML元素
$(“div#intro .head”) id=”intro” 的 <div>元素中所有class="head"的元素

所有元素、id、class、指定元素、多个class

选择器 实例 选取
* $(“*”) 所有元素
#id $(“#lastname”) id="lastname" 的元素
.class $(“.intro”) 所有 class="intro" 的元素
element $(“p”) 所有 <p> 元素
.class.class $(“.intro.demo”) 所有 class="intro" 且 class="demo" 的元素

首个、末个、偶数、奇数

选择器 实例 选取
:first $(“p:first”) 第一个 <p> 元素
:last $(“p:last”) 最后一个 <p> 元素
:even $(“tr:even”) 所有偶数 <tr> 元素
:odd $(“tr:odd”) 所有奇数 <tr> 元素

第几个、大于or小于index的、不为空的

选择器 实例 选取
:eq(index) $(“ul li:eq(3)”) 列表中的第四个元素（index 从 0 开始）
:gt(no) $(“ul li:gt(3)”) 列出 index 大于 3 的元素
:lt(no) $(“ul li:lt(3)”) 列出 index 小于 3 的元素
:not(selector) $(“input:not(:empty)”) 所有不为空的 input 元素

h1~h6标题、动画

选择器 实例 选取
:header $(“:header”) 所有标题元素 <h1> - <h6>
:animated 所有动画元素

包含、无子节点、隐藏的、可见的

选择器 实例 选取
:contains(text) $(“:contains(‘W3School’)”) 包含指定字符串的所有元素
:empty $(“:empty”) 无子（元素）节点的所有元素
:hidden $(“p:hidden”) 所有隐藏的 <p> 元素
:visible $(“table:visible”) 所有可见的表格

匹配选择的

选择器 实例 选取
s1,s2,s3 $(“th,td,.intro”) 所有带有匹配选择的元素

href：带href、href属性值比较

选择器 实例 选取
[attribute] $(“[href]”) 所有带有 href 属性的元素
[attribute=value] $(“[href=’#’]”) 所有 href 属性的值等于 “#” 的元素
[attribute!=value] $(“[href!=’#’]”) 所有 href 属性的值不等于 “#” 的元素
[attribute$=value] $(“[href$=’.jpg’]”) 所有 href 属性的值包含以 “.jpg” 结尾的元素

input：所有input、不同type类型的

选择器 实例 选取
:input $(“:input”) 所有 <input> 元素
:text $(“:text”) 所有 type=”text” 的 <input> 元素
:password $(“:password”) 所有 type=”password” 的<input>元素
:radio $(“:radio”) 所有 type=”radio” 的 <input>元素
:checkbox $(“:checkbox”) 所有 type=”checkbox” 的 <input> 元素
:submit $(“:submit”) 所有 type=”submit” 的 <input> 元素
:reset $(“:reset”) 所有 type=”reset” 的 <input> 元素
:button $(“:button”) 所有 type=”button” 的 <input> 元素
:image $(“:image”) 所有 type=”image” 的 <input> 元素
:file $(“:file”) 所有 type=”file” 的 <input> 元素

input状态的：激活、禁用、被选取、被选中

选择器 实例 选取
:enabled $(“:enabled”) 所有激活的 input 元素
:disabled $(“:disabled”) 所有禁用的 input 元素
:selected $(“:selected”) 所有被选取的 input 元素
:checked $(“:checked”) 所有被选中的 input 元素

jQuery遍历

祖先

parent()

返回被选元素的直接父元素（即只上一级元素，只会有一个父元素）

parents()

返回被选元素的所有祖先元素，一直到根元素<html>（包含<html>）

parentsUntil()

返回介于两个给定元素之间的所有祖先元素（不包含这两个给定的元素）

后代

children()

返回被选元素的所有直接子元素（即只下一级的元素，也可能会同时找到多个）

find()

返回被选元素的后代元素，一路向下直到最后一个后代（找底下所有嵌套元素中符合条件的元素，会同时找到多个）

同胞

同胞拥有相同的父元素

siblings()

返回被选元素的所有同胞元素

next()

返回被选元素的下一个同胞元素（只返回一个）

nextAll()

返回被选元素的所有跟随的同胞元素

nextUntil()

返回介于两个给定参数之间的所有跟随的同胞元素（就是两者之间的所有元素，不包括两者）

过滤

first()

返回被选元素的首个元素

last()

返回被选元素的最后一个元素（嵌套多层，会找到最底层的那个）

eq()

返回被选元素中带有指定索引号的元素

索引号从0开始

filter()

自定义一个标准。不匹配的会被从集合中删除，匹配的元素会返回。

not()

返回不匹配标准的所有元素（与filter()取相反的元素）
