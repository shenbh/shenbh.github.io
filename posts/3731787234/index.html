Windows-提升效率

Posted on 2020-01-17 | In 工具

清理缓存垃圾

运行（Win 键+R），输入cmd回车

  1. 输入 chkdsk，回车。等待清理垃圾
  2. 再输入sfc/scannow，回车等待扫描修复完成即可

需要用管理员权限运行：任务管理器中》文件》运行新任务》输入cmd，勾选“以系统管理权限创建此任务”》然后输入命令chkdsk回车，输入sfc/scannow回车

清理垃圾

运行（Win 键+R），输入cleanmgr，选择一个盘如 C 盘，“C 的磁盘清理”弹窗中勾选要清理的，点确定即可

提升windows运行速度

可选软件：系统提速精灵

卸载垃圾、无用软件

“开始–控制面板–添加删除程序”：找到要删除的程序，点卸载

有些流氓软件找不到，或卸载不掉总是提示被占用，推荐“Geek Uninstaller”（免费、无广告、速度快、体积小）。类似软件：CleanMyMac X（Mac用户）

限制开机自启动

（Windows10）Ctrl+Shift+Esc切换到“启动”选项卡。选中不想要开机启动项进行“禁用”

（Windows10以下）“运行”–输入“msconfig”回车，调出“系统配置菜单”将不需要启动的软件取消选中，点击确定按钮。

定时清理缓存

bat脚本内容看：看“bat相关–清理缓存”

定时功能看：底下“设置–创建定时任务”

关闭一些不必要的服务

右键单击桌面上的“此电脑”，选择“管理”，双击“服务和应用程序”，然后双击“服务”

  • Windows Search 更改称“手动”或“禁用”并停止
  • Windows Update 更改为“手动”或“禁用”并停止
  • Connected User Experiences and Telemetry 是微软的一个外围服务，用于收集一些信息
  • 四个”Diagnostic”服务 系统诊断服务，用于支持和执行系统诊断
  • HomeGroup Listener
  • HomeGroup Provider

还可以使用**DISM++**这款电脑管理工具进行更多的操作（windows系统管理百宝箱）：一键禁用win10系统更新、一键禁用Defender、垃圾清理、启动项管理、系统优化等

关闭休眠

“win+x”，选择“命令提示符（管理员）”或英文“Windows PowerShell（管理员）”

输入powercfg -h off

重启电脑，休眠功能即将关闭，也会释放它所占用的硬盘空间

关闭P2P分享服务

win+i，点击“更新与安全”

切换至“传递优化”选项卡，关闭“允许去其他电脑下载”即可

设置硬盘的部分为虚拟内存

电脑–属性–高级系统设置：切换到“高级”选项卡–点击性能“设置”–高级–虚拟内存“更改”：

  1. 去除“自动管理所有驱动器的分页文件大小”的勾选

  2. 选中c盘系统盘，设置无分页

  3. 设置虚拟内存到非系统盘，比如选中E盘，勾选“自定义大小”，初始值和最大值设置为2048，再点“设置”完成修改

  4. 重启电脑，生效

磁盘整理（清理磁盘碎片）

  • 系统自带

打开控制面板–系统和安全–管理工具–对你的驱动器进行碎片整理和优化

或者：打开“我的电脑”后，选择要做整理的盘符，“右键”它在“属性”里面选择“磁盘清理”即可

另：Win10自动清理垃圾功能：“设置–系统–存储”，开启“存储感知功能”可自动清理临时文件

如果是固态硬盘，则无需进行碎片整理

  • 分析磁盘：SpaceSniffer

内存不足，不知道被什么东西占用了空间

SpaceSniffer软件可以帮助分析

Win10每天自动清理临时文件

设置》系统》存储》打开“存储感知”，点击“更改详细设置”，并设置临时文件和下载文件的存在时间

把桌面从C盘挪出

  1. 在目标磁盘上新建个文件夹

  2. Windows键+E，弹出资管管理器，右键“桌面”选择“属性”，切换至“位置”选项卡，并点击“移动”。选择上一步新建的文件夹，点确定。

检查病毒和间谍软件

使用内置的Windows Defender或第三方引用程序，如开源的“Ty2y”杀毒软件，防止病毒木马影响系统速度

让 windows 从高性能模式上升到暴走模式

Win+X，打开Windows PwerShell，输入powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61

打开控制面板》电源选项》隐藏附加计划内多了“卓越性能”一项，选择它

使用性能疑难解答

打开控制面板并搜索“疑难解答”，在“系统和安全性”下运行疑难解答，它可能会找到一些导致速度下降的根本原因。那么我们可以根据提示进行一些优化操作。

更改视觉效果

找到系统的性能选项，选中“调整为最佳性能”的单选按钮。则将失去动态、阴影等视觉效果，但会让电脑运行更快。

关闭搜索索引

在“开始”按钮旁边的搜索框中键入“索引”，在结果列表顶部看到“索引选项”，点击它打开“索引选项”窗口，点击“修改”按钮，然后取消选中不想建立索引的位置，单击“高级”按钮决定应该和不应该索引哪些文件类型。

关闭提示和通知

Windows10的“提示”

打开设置，单击“系统–通知和操作”，在右侧，关闭在使用Windows时获取提示、技巧和建议”选项。

更改默认存储位置

Win10。设置》系统》存储。右侧找到“更多存储”，点击“更改新内容的保存位置”，把默认保存到C盘的都改成其他盘。

CPU太慢调整

Win+R，输入msconfig，在“系统配置”弹窗里，切换到“引导”页签，点击“高级选项”，设置处理器个数为最大。

解除下载速度限制

  1. Win+R，输入 gpedit.msc
  2. 找到计算机配置–Windows 设置–基于策略的 Qos，右键选择高级 Qos 设置，入站TCP流量页签内勾选“指定入站TCP 吞吐量级别”勾选级别 3，点击确定
  3. 找到计算机配置–管理模板–网络，双击打开 Qos 数据包计划程序，双击“限制可保留带宽”，勾选”已启用“，下面的数字改成”0“，点确定即可

设置加速启动Windows

电脑右键》属性》高级系统设置》系统属性，切换到“高级”，点击“启动和故障恢复”下的“设置”，弹出框中勾选“显示操作系统列表的时间”、“在需要时显示恢复选项的时间”，并且把这两个的值改成“0”
