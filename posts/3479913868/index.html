Jetpack-WorkManager

еҸ‘иЎЁдәҺ 2019-12-24 жӣҙж–°дәҺ 2021-07-15 еҲҶзұ»дәҺ Android
жң¬ж–Үеӯ—ж•°пјҡ 6.1k йҳ…иҜ»ж—¶й•ҝ ≈ 6 еҲҶй’ҹ

WorkManager

AndroidдёәеҗҺеҸ°жү§иЎҢд»»еҠЎжҸҗдҫӣдәҶеӨҡз§Қи§ЈеҶіж–№жЎҲпјҡJobSchedulerгҖҒLoaderгҖҒServiceзӯүгҖӮдҪҶе®ғд»¬еҸҜиғҪдјҡж¶ҲиҖ—еӨ§йҮҸзҡ„з”өйҮҸгҖӮAndroidеңЁиҖ—з”өй—®йўҳдёҠеҸҲеҒҡдәҶеҗ„з§Қе°қиҜ•пјҢд»ҺDozeеҲ°App StandbyгҖӮ

WorkManagerдёәеә”з”ЁзЁӢеәҸдёӯйӮЈдәӣдёҚйңҖиҰҒеҸҠж—¶е®ҢжҲҗзҡ„д»»еҠЎпјҲеҰӮеҗҺеҸ°жү§иЎҢд»»еҠЎпјүжҸҗдҫӣз»ҹдёҖзҡ„и§ЈеҶіж–№жЎҲпјҢд»ҘдҫҝеңЁи®ҫеӨҮз”өйҮҸе’Ңз”ЁжҲ·дҪ“йӘҢд№Ӣй—ҙиҫҫеҲ°дёҖдёӘиҫғеҘҪзҡ„е№іиЎЎгҖӮ

WorkManagerзҡ„3дёӘйҮҚиҰҒзү№зӮ№

  1. й’ҲеҜ№зҡ„жҳҜдёҚйңҖиҰҒеҸҠж—¶е®ҢжҲҗзҡ„д»»еҠЎгҖӮ

    дҫӢеҰӮпјҢеҸ‘йҖҒеә”з”ЁзЁӢеәҸж—Ҙеҝ—гҖҒеҗҢжӯҘеә”з”ЁзЁӢеәҸж•°жҚ®гҖҒеӨҮд»Ҫз”ЁжҲ·ж•°жҚ®зӯүпјҢз«ҷеңЁдёҡеҠЎйңҖжұӮзҡ„и§’еәҰпјҢиҝҷдәӣд»»еҠЎйғҪдёҚйңҖиҰҒз«ӢеҚіе®ҢжҲҗгҖӮеҰӮжһңжҲ‘д»¬иҮӘе·ұз®ЎзҗҶиҝҷдәӣд»»еҠЎпјҢйҖ»иҫ‘дјҡеҫҲеӨҚжқӮпјҢиӢҘAPIдҪҝз”ЁдёҚжҒ°еҪ“пјҢдјҡж¶ҲиҖ—еӨ§йҮҸз”өйҮҸгҖӮ

  2. дҝқиҜҒд»»еҠЎдёҖе®ҡдјҡиў«жү§иЎҢгҖӮ

    WorkManagerиғҪдҝқиҜҒд»»еҠЎдёҖе®ҡдјҡиў«жү§иЎҢпјҢеҚідҪҝеә”з”ЁзЁӢеәҸеҪ“еүҚдёҚеңЁиҝҗиЎҢдёӯпјҢи®ҫзҪ®еңЁи®ҫеӨҮйҮҚеҗҜиҝҮеҗҺпјҢд»»еҠЎд»Қз„¶дјҡеңЁйҖӮеҪ“зҡ„ж—¶еҲ»иў«жү§иЎҢгҖӮиҝҷжҳҜеӣ дёәWorkManagerжңүиҮӘе·ұзҡ„ж•°жҚ®еә“пјҢе…ідәҺд»»еҠЎзҡ„жүҖжңүдҝЎжҒҜе’Ңж•°жҚ®йғҪдҝқеӯҳеңЁиҜҘж•°жҚ®еә“дёӯгҖӮеӣ жӯӨпјҢеҸӘиҰҒд»»еҠЎдәӨз»ҷдәҶWorkManagerпјҢе“ӘжҖ•еә”з”ЁзЁӢеәҸеҪ»еә•йҖҖеҮәпјҢжҲ–иҖ…и®ҫеӨҮиў«йҮҚеҗҜпјҢWorkManagerд»ҚиғҪдҝқиҜҒе®ҢжҲҗдҪ дәӨз»ҷе®ғзҡ„д»»еҠЎгҖӮ

  3. е…је®№иҢғеӣҙе№ҝгҖӮ

    WorkManagerжңҖдҪҺиғҪе…је®№API Level 14пјҢе№¶дё”дёҚйңҖиҰҒдҪ зҡ„и®ҫеӨҮе®үиЈ… Google Play ServicesгҖӮеӣ жӯӨпјҢдёҚз”ЁиҝҮдәҺжӢ…еҝғе…је®№жҖ§й—®йўҳпјҢеӣ дёә API Level 14 е·Із»ҸиғҪе…је®№еҮ д№Һ100%зҡ„и®ҫеӨҮгҖӮ

WorkManagerзҡ„е…је®№ж–№жЎҲ

WorkManagerдҫқжҚ®и®ҫеӨҮзҡ„жғ…еҶөдјҡйҖүжӢ©дёҚеҗҢзҡ„жү§иЎҢж–№жЎҲгҖӮеңЁAPI Level 23д»ҘдёҠзҡ„и®ҫеӨҮдјҡйҖҡиҝҮ JobScheduler е®ҢжҲҗд»»еҠЎпјӣеңЁ API Level 23д»ҘдёӢзҡ„и®ҫеӨҮдёӯдјҡйҖҡиҝҮ AlarmManager е’Ң BroadcastReceiver з»„еҗҲжқҘе®ҢжҲҗд»»еҠЎгҖӮиҝҷдёӨз§Қж–№жЎҲпјҢд»»еҠЎжңҖз»ҲйғҪжҳҜдәӨз”ұ Executor жқҘжү§иЎҢгҖӮ

жіЁж„ҸпјҡWorkManagerдёҚжҳҜдёҖз§Қж–°зҡ„е·ҘдҪңзәҝзЁӢпјҢе®ғзҡ„еҮәзҺ°дёҚжҳҜдёәдәҶжӣҝд»Је…¶д»–зұ»еһӢзҡ„е·ҘдҪңзәҝзЁӢгҖӮе·ҘдҪңзәҝзЁӢйҖҡеёёз«ӢеҚіиҝҗиЎҢпјҢе№¶еңЁд»»еҠЎе®ҢжҲҗеҗҺз»ҷз”ЁжҲ·еҸҚйҰҲгҖӮиҖҢWorkManagerдёҚжҳҜеҚіж—¶зҡ„пјҢдёҚиғҪдҝқиҜҒд»»еҠЎиғҪз«ӢеҚіеҫ—еҲ°жү§иЎҢгҖӮ

WorkManagerзҡ„еҹәжң¬дҪҝз”Ёж–№жі•

еңЁ app/build.gradleдёӯж·»еҠ  WorkManager жүҖйңҖзҡ„дҫқиө–

дҪҝз”Ё Worker зұ»е®ҡд№үд»»еҠЎ

еҒҮи®ҫиҰҒе®һзҺ°е°Ҷж—Ҙеҝ—дёҠдј еҲ°жңҚеҠЎеҷЁзҡ„йңҖжұӮгҖӮж–°е»әдёҖдёӘеҗҚдёәUploadLogWorkerзҡ„зұ»пјҢиҜҘзұ»з»§жүҝиҮӘWorkerзұ»пјҢе№¶дё”иҰҶзӣ–doWork()ж–№жі•пјҢжүҖжңүйңҖиҰҒеңЁд»»еҠЎдёӯжү§иЎҢзҡ„д»Јз ҒйғҪеңЁиҜҘж–№жі•дёӯиҝӣиЎҢзј–еҶҷгҖӮ

doWork() ж–№жі•жңү3дёӯзұ»еһӢзҡ„иҝ”еӣһеҖј

  • иӢҘжү§иЎҢжҲҗеҠҹпјҢеҲҷиҝ”еӣһResult.success()гҖӮ
  • иӢҘжү§иЎҢеӨұиҙҘпјҢеҲҷиҝ”еӣһResult.failure()гҖӮ
  • иӢҘйңҖиҰҒйҮҚж–°жү§иЎҢпјҢеҲҷиҝ”еӣһResult.retry()гҖӮ

дҪҝз”ЁWorkRequestй…ҚзҪ®д»»еҠЎ

й…ҚзҪ®д»»еҠЎе°ұжҳҜеңЁе‘ҠиҜүзі»з»ҹпјҢд»»еҠЎдҪ•ж—¶иҝҗиЎҢеҸҠеҰӮдҪ•иҝҗиЎҢгҖӮ

  1. и®ҫзҪ®д»»еҠЎи§ҰеҸ‘жқЎд»¶гҖӮ

    дҫӢеҰӮпјҢжҲ‘д»¬еҸҜд»Ҙи®ҫзҪ®еҪ“и®ҫеӨҮеӨ„дәҺе……з”өпјҢзҪ‘з»ңе·ІиҝһжҺҘпјҢдё”з”өжұ з”өйҮҸе……и¶ізҡ„зҠ¶жҖҒдёӢпјҢжүҚи§ҰеҸ‘д»»еҠЎгҖӮ

  2. е°Ҷд»»еҠЎи§ҰеҸ‘жқЎд»¶и®ҫзҪ®еҲ°WorkRequestгҖӮ

    WorkRequestжҳҜдёҖдёӘжҠҪиұЎзұ»пјҢе®ғжңүдёӨз§Қе®һзҺ°ж–№ејҸвҖ”вҖ”OneTimeWorkRequestе’ҢPeriodicWorkRequestпјҢеҲҶеҲ«еҜ№еә”зҡ„жҳҜдёҖж¬ЎжҖ§д»»еҠЎе’Ңе‘ЁжңҹжҖ§д»»еҠЎгҖӮ

  3. и®ҫзҪ®е»¶иҝҹжү§иЎҢд»»еҠЎгҖӮ

    еҒҮи®ҫжІЎжңүи®ҫзҪ®и§ҰеҸ‘жқЎд»¶пјҢжҲ–иҖ…жүҖи®ҫзҪ®зҡ„и§ҰеҸ‘жқЎд»¶жӯӨеҲ»з¬ҰеҗҲзі»з»ҹзҡ„жү§иЎҢиҰҒжұӮпјҢжӯӨж—¶пјҢзі»з»ҹжңүеҸҜиғҪдјҡз«ӢеҲ»жү§иЎҢиҜҘд»»еҠЎгҖӮдҪҶеҰӮжһңдҪ еёҢжңӣиғҪеӨҹе»¶иҝҹд»»еҠЎзҡ„жү§иЎҢпјҢйӮЈд№ҲеҸҜд»ҘйҖҡиҝҮsetInitialDelay()ж–№жі•пјҢеҜ№д»»еҠЎиҝӣиЎҢе»¶еҗҺгҖӮ

  4. и®ҫзҪ®жҢҮж•°йҖҖйҒҝзӯ–з•ҘгҖӮ

    еҒҮеҰӮWorkerзәҝзЁӢзҡ„жү§иЎҢеҮәзҺ°дәҶејӮеёёпјҢеҰӮжңҚеҠЎеҷЁе®•жңәгҖӮдҪ еҸҜиғҪеёҢжңӣиҝҮдёҖж®өж—¶й—ҙеҗҺпјҢйҮҚиҜ•иҜҘд»»еҠЎгҖӮйӮЈд№ҲеҸҜд»ҘеңЁWorkerзҡ„doWork()ж–№жі•дёӯиҝ”еӣһResult.retry()пјҢзі»з»ҹдјҡжңүй»ҳи®Өзҡ„жҢҮж•°йҖҖйҒҝзӯ–з•ҘжқҘеё®дҪ йҮҚиҜ•д»»еҠЎпјҢд№ҹеҸҜд»ҘйҖҡиҝҮsetBackoffCriteria()ж–№жі•жқҘиҮӘе®ҡд№үжҢҮж•°йҖҖйҒҝзӯ–з•ҘгҖӮ

  5. дёәд»»еҠЎи®ҫзҪ®tagж ҮзӯҫгҖӮ

    и®ҫзҪ®tagж ҮзӯҫеҗҺпјҢдҪ е°ұеҸҜд»Ҙ

    йҖҡиҝҮиҜҘж Үзӯҫи·ҹиёӘд»»еҠЎзҡ„зҠ¶жҖҒпјҡWorkManager.getWorkInfosByTagLiveDataпјҲString tagпјүпјӣ

    д№ҹеҸҜд»ҘеҸ–ж¶Ҳд»»еҠЎпјҡWorkManager.cancelAllWorkByTagпјҲString tagпјүгҖӮ

е°Ҷд»»еҠЎжҸҗдәӨз»ҷзі»з»ҹ

е°Ҷд»»еҠЎй…ҚзҪ®еҘҪд№ӢеҗҺпјҢйңҖиҰҒе°Ҷе…¶жҸҗдәӨз»ҷзі»з»ҹпјҢWorkManager.enqueue()ж–№жі•з”ЁдәҺе°Ҷй…ҚзҪ®еҘҪзҡ„WorkRequestдәӨз»ҷзі»з»ҹжқҘжү§иЎҢгҖӮ

и§ӮеҜҹд»»еҠЎзҡ„зҠ¶жҖҒ

д»»еҠЎеңЁжҸҗдәӨз»ҷзі»з»ҹеҗҺпјҢеҸҜд»ҘйҖҡиҝҮWorkInfoиҺ·зҹҘд»»еҠЎзҡ„зҠ¶жҖҒгҖӮWorkInfoеҢ…еҗ«д»»еҠЎзҡ„idгҖҒtagгҖҒWorkerеҜ№иұЎдј йҖ’иҝҮжқҘзҡ„outputDataпјҢд»ҘеҸҠд»»еҠЎеҪ“еүҚзҡ„зҠ¶жҖҒгҖӮжңү3з§Қж–№ејҸеҸҜд»Ҙеҫ—еҲ°WorkInfoеҜ№иұЎгҖӮ

  • WorkManager.getWorkInfosByTag()
  • WorkManager.getWorkInfoById()
  • WorkManager.getWorkInfosForUniqueWork()

еҰӮжһңеёҢжңӣе®һж—¶иҺ·зҹҘд»»еҠЎзҡ„зҠ¶жҖҒпјҢиҝҷ3дёӘж–№жі•иҝҳжңүеҜ№еә”зҡ„LiveDataж–№жі•гҖӮ

  • WorkManager.getWorkInfosByTagLiveData()
  • WorkManager.getWorkInfoByIdLiveData()
  • WorkManager.getWorkInfosForUniqueWorkLiveData()

йҖҡиҝҮLiveDataпјҢжҲ‘д»¬дҫҝеҸҜд»ҘеңЁд»»еҠЎзҠ¶жҖҒеҸ‘з”ҹеҸҳеҢ–ж—¶ж”¶еҲ°йҖҡзҹҘгҖӮ

еҸ–ж¶Ҳд»»еҠЎ

дёҺи§ӮеҜҹд»»еҠЎзұ»дјјпјҢжҲ‘д»¬д№ҹеҸҜд»Ҙж №жҚ®idжҲ–tagеҸ–ж¶ҲжҹҗдёӘд»»еҠЎпјҢжҲ–иҖ…еҸ–ж¶ҲжүҖжңүзҡ„д»»еҠЎгҖӮ

WorkManagerдёҺWorkerд№Ӣй—ҙзҡ„еҸӮж•°дј йҖ’

WorkManagerйҖҡиҝҮsetInputData()ж–№жі•еҗ‘Workerдј йҖ’ж•°жҚ®гҖӮж•°жҚ®зҡ„дј йҖ’йҖҡиҝҮDataеҜ№иұЎжқҘе®ҢжҲҗгҖӮйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢDataеҸӘиғҪз”ЁдәҺдј йҖ’дёҖдәӣе°Ҹзҡ„еҹәжң¬зұ»еһӢзҡ„ж•°жҚ®пјҢдё”ж•°жҚ®жңҖеӨ§дёҚиғҪи¶…иҝҮ10KBгҖӮ

WorkerйҖҡиҝҮgetInputData()ж–№жі•жҺҘж”¶ж•°жҚ®пјҢе№¶еңЁд»»еҠЎе®ҢжҲҗеҗҺпјҢеҗ‘WorkManagerиҝ”еӣһж•°жҚ®гҖӮ

WorkManagerйҖҡиҝҮLiveDataеҫ—еҲ°д»ҺWorkerиҝ”еӣһзҡ„ж•°жҚ®гҖӮ

е‘ЁжңҹжҖ§д»»еҠЎPeriodicWorkRequest

еүҚйқўжҸҗеҲ°иҝҮпјҢWorkRequestжңүдёӨз§Қе®һзҺ°ж–№ејҸпјҡOneTimeWorkRequestе’ҢPeriodicWorkRequestпјҢеҲҶеҲ«еҜ№еә”зҡ„жҳҜдёҖж¬ЎжҖ§д»»еҠЎе’Ңе‘ЁжңҹжҖ§д»»еҠЎгҖӮдёҖж¬ЎжҖ§д»»еҠЎеңЁд»»еҠЎжҲҗеҠҹжү§иЎҢеҗҺпјҢдҫҝеҪ»еә•з»“жқҹгҖӮиҖҢе‘ЁжңҹжҖ§д»»еҠЎеҲҷдјҡжҢүз…§и®ҫе®ҡзҡ„ж—¶й—ҙе®ҡжңҹжү§иЎҢгҖӮдәҢиҖ…дҪҝз”Ёиө·жқҘжІЎжңүеӨӘеӨ§е·®еҲ«гҖӮйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢе‘ЁжңҹжҖ§д»»еҠЎзҡ„й—ҙйҡ”ж—¶й—ҙдёҚиғҪе°‘дәҺ15еҲҶй’ҹгҖӮ

д»»еҠЎй“ҫ

еҰӮжһңжңүдёҖзі»еҲ—зҡ„д»»еҠЎйңҖиҰҒжҢүйЎәеәҸжү§иЎҢпјҢйӮЈд№ҲеҸҜд»ҘеҲ©з”ЁWorkManager.beginWith().then().then()...enqueue()зҡ„ж–№ејҸжһ„е»әд»»еҠЎй“ҫгҖӮдҫӢеҰӮпјҢеңЁдёҠдј ж•°жҚ®д№ӢеүҚпјҢеҸҜиғҪйңҖиҰҒе…ҲеҜ№ж•°жҚ®иҝӣиЎҢеҺӢзј©гҖӮ

еҒҮи®ҫйҷӨдәҶеҺӢзј©ж•°жҚ®пјҢиҝҳйңҖиҰҒжӣҙж–°жң¬ең°ж•°жҚ®гҖӮеҺӢзј©ж•°жҚ®дёҺжӣҙж–°жң¬ең°ж•°жҚ®дәҢиҖ…жІЎжңүе…ҲеҗҺйЎәеәҸзҡ„еҢәеҲ«пјҢдҪҶдёҺдёҠдј ж•°жҚ®еӯҳеңЁе…ҲеҗҺйЎәеәҸгҖӮ

еҒҮи®ҫжңүжӣҙеӨҚжқӮзҡ„д»»еҠЎй“ҫпјҢйӮЈд№ҲиҝҳеҸҜд»ҘиҖғиҷ‘дҪҝз”ЁWorkContinuation.combine()ж–№жі•пјҢе°Ҷд»»еҠЎй“ҫз»„еҗҲиө·жқҘдҪҝз”ЁгҖӮеңЁдёӢйқўзҡ„д»Јз ҒдёӯпјҢд»»еҠЎзҡ„жү§иЎҢйЎәеәҸдёәAгҖҒBгҖҒCгҖҒDгҖҒEгҖӮд»»еҠЎй“ҫжөҒзЁӢеҰӮеӣҫ7-2жүҖзӨәгҖӮ
