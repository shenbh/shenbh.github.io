科目二基础

安全带操作方法

视频链接：https://www.bilibili.com/video/BV1H4411G7gB?p=6

  1. 左手拉住安全带向前拉动，右手拉住锁扣
  2. 双手交替拉动，调整安全带长度
  3. 将锁扣插入卡槽，听见响声后，拉动锁扣确认插入正常，放开双手完成操作

方向盘操作方法

视频链接：https://www.bilibili.com/video/BV1H4411G7gB?p=12&spm_id_from=pageDriver

方向盘操作基本原则：

  1. 手握方向盘，双手要三九点平放。
  2. 转动方向盘，双手交叉，超过180度时，必需换手。
  3. 手指不要进入方向盘内侧，因道路情况多种多样，如果碾压到较大的石块，方向盘可能会反转，双手容易收到伤害。

离合器的操作方法

视频链接：https://www.bilibili.com/video/BV1H4411G7gB?p=12&spm_id_from=pageDriver

  1. 悬空松离合器的方法：

    左脚慢慢松离合器，感知到车辆抖动时，上下微调离合器高低，幅度要小频率要慢，控制抖动感在抖动与不抖动之间达到车辆控制在最慢车速

  2. 脚后跟做支点的方法：

    左脚慢慢松离合器，左脚后跟要跟随抬起并慢慢向后滑动，感知到车辆抖动时，用脚腕力量上下微调离合器高低，控制抖动感在抖动与不抖动之间达到车辆控制在最慢车速。

驻车制动器（手刹）操作方法

视频链接：https://www.bilibili.com/video/BV1H4411G7gB?p=14&spm_id_from=pageDriver

在车辆停止的状态下使用

  1. 拉起手刹

    五指环握手柄，快速拉起，仪表盘手刹指示灯亮起

  2. 放下手刹

    往上轻提手柄，大拇指按下锁扣，慢慢放下手柄，仪表盘指示灯熄灭

注意：拉起手刹时，不要按下锁扣，防止手刹未被拉紧。

图解科二

左右倒库

右倒库：

  1. 系好安全带，调整位置，调整左后视镜（后门把手在后视镜靠窗一侧的三分之一处，能看到后轮地面）；

  2. 踩刹车离合，空档点火，放手刹，挂一档，松刹车，慢抬离合，缓慢前进；

  3. 看左后视镜观察后轮出10cm左右（此时雨刮器底部接近前面的横线）快速右打满；到车头快要平行时快速左打满（车头回正）让前轮驶出虚线，虚线与肩线平齐时停住；

  4. 倒车到肩线与左侧红线平齐（或者红线在车把手中间位置时）右打满；（正常情况）慢慢退到左后视镜看到直角，左打半圈，待车轮进入直角区域，方向盘（左打一圈）回正；（异常情况）如果距离大于30cm则继续左打微调，如果小于30cm那么还是右打满状态，待车轮进入直角区域慢慢调整。

    注意：后退的时候红线要在车把手那就开始右打满，否则太晚的话就没有调整的空间了，太早的话也比较难调整。

  5. 后车轮的轴与地面的红线在直线上停住。

上车准备

视频链接：https://www.bilibili.com/video/BV1H4411G7gB?p=1

操作要求

  1. 起步前要调整座椅

  2. 调整驾驶姿势

  3. 调整后视镜

  4. 系好安全带

  5. 发动机点火

  6. 打开左转向灯

  7. 查看左右路况

  8. 挂一档，松手刹，起步

评判标准

  1. 未系安全带，不合格
  2. 不使用或错误使用转向灯，扣10分

详情解析

  1. 步骤一：调整座椅

    调整到自己习惯的合适位置

  2. 步骤二：调整驾驶姿势

    身体尽量坐直，臀部与座椅贴实

  3. 步骤三：调整后视镜

    左后视镜调至方便看到“左后门把手和左后车轮”的位置；

    右后视镜调至到最下。驾驶位看到“右前门把手”在“右后视镜中间“位置左右

  4. 步骤四：系好安全带，发动机点火，打开左转向灯，查看左右路况，挂一档，松手刹，起步

左右倒库

视频链接：https://www.bilibili.com/video/BV1H4411G7gB?p=2&spm_id_from=pageDriver

（以下，第一倒为右倒，也有第一倒为左倒的）

考试要求：

  1. 挂一档低速进入考试区域。两个前轮均轧过控制线后停车。
  2. 右倒库车正停车，前进向左驶出车库，两个前轮均轧过控制线后停车。
  3. 左倒库车正停车。前进向右驶出车库，回到起始点。

PS：车辆进退途中不得停车；项目完成时间不能超过210秒（3分半）。

评判标准

  1. 全程限210秒内完成，超过则不合格（新规）
  2. 中途停车，每次停车扣5分（新规）
  3. 不按规定路线顺序行驶，不合格
  4. 倒库不入，不合格；车身出线，不合格
  5. 倒车前，两个前轮未通过库区控制线，不合格

详情解析

  1. 步骤一：挂一档，低速行驶到考试区域，调整方向，使（左侧）车身距库边白线1.5m～1.8m，当肩线与控制线重合时停车

  2. 步骤二：停车超过3秒后，挂倒档，低速行驶，当左后视镜下沿与控制线重合时，方向盘迅速向右打满

  3. 步骤三：观察右后视镜，根据车身与库区左前角线距离做调整入库，当大于30cm时，向右打方向盘；小于30cm时，向左打方向盘

  4. 步骤四：车身与库线即将平行时，方向盘向左回正（亦可观察左后视镜，当左库底角出现在后视镜中时，回正方向）；观察左后视镜，当左后视镜下沿遮住库区左前方横线时，停车；控制车尾与库区白线距离在35cm左右。

  5. 步骤五：停车超过3秒后，挂一档出库。观察左后视镜：当左后轮与库位前角平齐时，迅速向左打满方向盘，行驶到控制线，当肩线与控制线平齐时，停车。

  6. 步骤六：停车超过3秒后，挂倒档，低速行驶。通过左后视镜观察车身与库区左前角线距离做调整入库：当大于30cm时，向左打方向盘；当小于30cm时，向右打方向盘。

  7. 步骤七：当车身与库位平行时，方向盘回正；观察左后视镜，当左后视镜下沿遮住库区左前方横线时，停车。

  8. 步骤八：停车超过3秒后，挂一档出库。观察引擎盖与库区白线重合时，向右打满方向盘，驶出考试区域。

驾校笔记

科目二：

  1. 科二顺序：侧方位停车，定点爬坡，直角转弯，走S弯，右倒库，左倒库，开到起点结束。
  2. 四个灯：从起点起步打左灯；从侧方位开出来打左灯；在坡上定点处起步打左灯；过直角过程前打灯（左打死过直角的时候灭灯）
  3. 教练车：8091、8032、5026、捷达车6227、捷达车7227
  4. 直角曲线不限时

科二只有Ｓ弯才允许车身扫线，其他的扫线直接挂。

停顿扣5分。熄火扣15分。一个灯10分。

侧方停车

  1. 准备：调整好位置

    打左转灯、挂1档，放手刹

  2. 前进：前进过程注意车标，让车身和黄线距离30cm左右，右雨刮器尖尖位置（有胶布的位置）跟前面竖直的黄线标对齐，开到车盖盖住黄线，停住

  3. 入库：先右打满，挂倒档，看左后视镜，看到库角回正

  4. 回正后，后退到左后车轮触碰到虚线左打满，再继续后倒（可以瞄下左后视镜看是否进车库了）后倒到车前盖左线和黄线外侧平齐，停住

  5. 出库：（左打满状态）打左转灯，挂1档前进，看车前盖左前端碰黄线，回正，看车标触碰横线右打一圈，往前开，过程中注意车身回正，开到后轮出线，结束。

  6. 训练过程中需要后倒原点。后倒过程看右后视镜，保持后车轮眉与横线有两指宽度，倒到横杆和肩线平齐停止，挂空档拉手刹换人。

半坡定点

  1. 侧方后前进过程：调整车身与右侧距离20-30公分（看中标）（过30cm扣10分，过50cm不及格）我，可以在坡底停。（等坡上没车了再爬坡）
  2. 开始爬坡过程：挂一档，慢抬离合，爬坡到（坐正，背靠椅背）肩膀与白线平齐停住（标准是车头在黄线中间）。（车头不在中间黄线扣10分，过前后俩黄线不及格）
  3. 定点后起步过程：拉手刹隔半秒放手刹，打左灯。刹车踩住，慢抬离合到车身抖动的厉害的时候，离合不动慢抬刹车，前进。
  4. 过坡停车：过坡到车头朝下的时候停下。
  5. 下坡过程：离合踩到底，抬一点点刹车，慢慢下坡，注意调整右侧距离30公分左右。

接着直角转弯

直角

  1. 准备：下坡，到坡底，停住（一停二看三通过）。
  2. 进直角前打左灯（要按住灯，不然调整方向盘的话它会自动变回去）
  3. 拐过直角：到肩膀与线平齐的时候左打死（关灯（唯一一次）），看左侧车身距离20-30公分（保证左边不压直角）。
  4. 过直角后回正。直角结束，注意不能停车，接着s弯。

注意：直角和s弯过程不能停车

s弯

  1. 左大灯中间碰到前面的黄线的时候，左打一圈
  2. 左弯走的过程：看灯贴着右线。到左车盖碰到黄线回正，再往前一点到车标碰到黄线右打一圈。
  3. 右弯走过程：右转弯过程保持左前轮与线50公分左右。
  4. 出s弯：到肩膀过直角左打死，拐过直角，s弯结束。

右左倒库

  1. 进库过程：车窗中间过角的时候左打死进库（车身与线的距离没有要求，不要压线即可）
  2. 右倒前：车轮过第二条黄色虚线中间（看雨刮器）停住。
  3. 接下来就是右倒库，然后左倒库。
  4. 左倒结束后出库：车头过右侧黄色虚线，开出去到起点位置就结束了。（注意过程中不要压线）

右倒库

  1. 准备：看左右后视镜，要能从左后视镜看到左后车门把手在镜子中靠车身位置1/3处。从右后视镜能看到库底。
  2. 右出库：前进，左后轮与库角平齐时右打满；
  3. 出右虚线：到车头快要平行时车头快速回正，让前轮驶出虚线，虚线与肩线平齐时停住；
  4. 右倒库：（车身是正的）倒车到肩线与左侧红线平齐（或者红线在车把手中间位置时。在平台那边是过车把20cm）右打满；慢慢退到左后视镜看到库角，左打半圈，如果距离大于30cm则继续左打微调，如果小于30cm那么维持左打满状态，微调入库。
  5. 到左后轮与地上的线平齐停车。

左倒库

方法一（教练教法）：

  1. 左出库：往前开，看左前轮（看不到的看轮眉）距离边线10cm。到左后轮与库角平齐，迅速左打满。
  2. 保持左打满，行进到车身与边线平行时，往右回1/4，左前车轮出左侧虚线20cm停住。
  3. 左倒库：保持方向盘（刚回了1/4的状态）开始后退，后退到车身与边线平行时，迅速左打满。
  4. 入库：看左后视镜，调整车身左侧与库角有30cm左右来入库。到车身与停车位的边线平行时回正（这个过程可以探头看下）
  5. 到左后轮与地上的线平齐停车。

方法二：

  1. 左出库：往前开，看肩膀与左侧库角平齐，左打一圈半，保持着出左侧虚线，车轮过虚线20cm停住。
  2. 左倒库：（保持左打一圈半状态）倒车，观察左后视镜，微调车身与库角有30cm左右。到车身与停车位的边线平行时回正（这个过程可以探头看下）
  3. 到左后轮与地上的线平齐停车。
