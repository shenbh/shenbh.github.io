Pycharm相关

发表于 2020-06-28 更新于 2021-04-08 分类于 工具
本文字数： 3.2k 阅读时长 ≈ 3 分钟

设置

https://mp.weixin.qq.com/s/qfox-HWnobCrCau98zM9bg

设置指定编译器

File–Settings–Project Interpreter：右侧选择一个python.exe

去掉一些检测

法一：Pycharm右下角的按钮，调整预警等级

法二（不推荐）：File–Settings–Editor–Inspections：勾掉即不检测

设置背景图片

File–Settings–Appearance & Behavior：点击按钮Background Image…设置图片

pycharm快捷键

  • 注释：选中后 Ctrl + /
  • 定位：Ctrl + 鼠标左键
  • 缩进：Tab / Tab+shift
  • 换行：shift + enter
  • 编译：Ctrl+shift+F10
  • 查看路径：ctrl+l (或在终端输入pwd)

插件推荐

插件名称 说明
CodeGlance 在右侧预览代码
TabNine 代码智能提示

使用技巧

代码排版，自动PEP8

  1. 安装工具
  1. 在 Pycharm 里导入这个工具

  1. 使用

在代码页面右键，选择External Tools–>AutoPep8

误删文件，一秒找回

Pycharm的Local History，点击子项Show History找到记录，点右键选择Revert

代码模板，效率编码

  • 新建文件时可以初始化文件

settings–》File and Code Templates–》Files 的 Python Script

  • ctrl+J可以调出预设的模板（Command+J）

使用书签，快速定位

ctrl+B（Command+B）

例如：给源码某位置加书签1，ctrl+1就能直接跳到这个位置

不加标记的话，就是匿名的，需要shift+F11列出所有书签来找

调试技巧

  • 断点后右键选择Evaluate Expression可以查看变量值

  • 断点后点击Show Python Prompt可以查看并操作所有变量（在Edit Configurations...中可以默认打开这个开关：勾选Show command line afterwards，但要debug运行）

指定参数执行脚本

如命令行中运行脚本的命令是这样

那么进入设置面板，在Script parameters中填入参数init --local即可

同样在设置中可以切换解释器：

Settings–》Project：VMP-Code–》Project Interpreter–》Project Interpreter选择切换

搜索时过滤测试文件

Find in Path的弹窗中勾选File mask:，填入!test*即可过滤测试文件

关闭烦人的灯泡提示

红灯泡，一般是语法问题，如果不处理会影响代码运行。

黄灯泡，只是提示，提示代码不规范等，并不会影响程序的运行。

去除灯泡提示（Pycharm2018版本），Settings–》搜索intention bulb，不勾选即可

关闭碍眼的波浪线

Pycharm右下角的头像点开，选择Syntax级别即可

一键进行代码性能分析

Run–》Profile ‘mytest’

  1. 性能统计界面由Name、Call Count、Time(ms)、Own Time(ms)组成的表格

  2. 点击Call Graph（调用关系图）界面直观展示了各函数直接的调用关系、运行时间和时间百分比

    1. 箭头表示调用关系，由调用者指向被调用者
    2. 举行的左上角显示模块或者函数的名称，右上角显示被调用的次数
    3. 举行中间显示运行时间和时间百分比
    4. 举行的颜色表示运行时间或者时间百分比大小的趋势：红色 > 黄绿色 > 绿色

使用Git做版本控制

VCS–》Checkout from Version Control–》Git

Tab轻松转空格

团队中有人喜欢用tab做缩进，有人喜欢用四个空格做缩进

旧文件：同一个Python文件模块中，tab和四个空格缩进两种缩进风格不能共存，Pycharm中由自动检测源文件的缩进风格，开启自动检测 Settings--》Editor--》Code Style--》勾选Detect and use existing file indents for editing和Show notifications about detected indents

新文件：Settings--》Editor--》Code Style--》Python，勾选Use tab character就会在新建文件时使用Tab进行缩进，反之使用四个空格进行缩进

源码文档，快速预览

  • 技巧：Ctrl+鼠标左键（Command+鼠标左键）可实现函数跳转查看源码

  • 技巧：快速查看函数文档（在函数的开头处，勇士三个引号包含的内容，叫做函数文档（DocString））Ctrl+q（Mac的未找到）

  • 技巧：快速预览源代码Ctrl+Shift+i（Mac：Command+Shift+i），可快速预览类或函数的代码逻辑

快速定位到错误行

F2和Shift + F2可以快速定位到错误行

快速查看最近的修改

Alt+Shift+C（Mac：option+Shift+C）

静态代码分析检查

右键项目的文件夹选择Inspect Code...

全方位无死角精准定位

给PyCharm装上ideaVim，给Chrome装上Vimium

  • 精准定位到文件：Ctrl+Shift+N（Mac：Command+Shift+N）
  • 精准定位到类：Ctrl+N（Mac：Command+N）
  • 精准定位到符号：类的所有成员（函数、变量等）Ctrl+Alt+Shift+N（Mac：option+Shift+Command+N）
  • 精准定位到文件结构：文件结构包括类、函数、变量 Ctrl+F2（Mac：Command+F2）
  • 精准定位到某行：Ctrl+G（Mac：Command+G）

利用TODO

调出TODO面板 Alt+6（Mac：Command+6）
